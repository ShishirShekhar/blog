# blog

## 📃 Intro

🎯 Goal: This website is a blog website

🎖 Feature: 
- You create a blog.
- You can edit you blogs.
- You check all the blogs on the website.


## ‼ Dependencies
    "body-parser": "^1.19.1",
    "dotenv": "^14.3.2",
    "ejs": "^3.1.6",
    "express": "^4.17.2",
    "mongoose": "^6.1.8"


## 👩‍💻 Install Dependencies
> npm i


## 🏃‍♂️ Run app
> node app.js
